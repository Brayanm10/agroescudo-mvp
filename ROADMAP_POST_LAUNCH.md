# Roadmap Post Launch - AgroEscudo

## P1

- UX completa de signup, invitacion y recuperacion en web y mobile.
- Sala de Control fullscreen para pantallas de planta.
- Upload real S3 con URLs firmadas.
- Email Resend real con plantillas.
- FCM para alertas push.
- WhatsApp/Telegram reales fuera de dry-run.

## P2

- LLM opcional para AgroAsistente con guardrails y auditoria.
- Analitica postcosecha avanzada.
- Integraciones ERP o BI bajo demanda.
- App store release cuando el piloto lo justifique.

## Fuera De Alcance

- Blockchain.
- Marketplace.
- Scoring financiero.
- Seguros o banca.
- Certificacion oficial automatica.
